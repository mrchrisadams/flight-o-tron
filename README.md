# flight-o-tron

A Vue App, designed to let you see the flights you've taken, take measures to reduce, or see how policy might affect you, so you can plan accordingly.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build

```
